# Azure Data Factory - Blob to Blob Data Copy

## Objective

Copy a CSV file from one Azure Blob Storage container to another using Azure Data Factory.

## Azure Services Used

* Azure Resource Group
* Azure Storage Account
* Azure Blob Storage
* Azure Data Factory
* Azure IAM
## Workflow

1. Create Resource Group
2. Create Storage Account
3. Create Source and Destination Containers
4. Upload Source CSV File
5. Create Azure Data Factory
6. Configure Linked Service
7. Create Source and Destination Datasets
8. Create Pipeline
9. Add Get Metadata Activity
10. Add Copy Data Activity
11. Execute Pipeline
12. Iam roles assigned 

## Result

The file was successfully copied from the source container to the destination container using Azure Data Factory.
